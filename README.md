# 🍜 Food Culture & Emotion Dashboard

## 🌍 Overview

The **Food Culture & Emotion Dashboard** is a data science project that analyzes food reviews using Natural Language Processing (NLP) to understand **sentiments and emotions across different cuisines and regions**.

This project transforms raw textual reviews into meaningful insights through interactive visualizations and AI models.

---

## 🚀 Features

* 🔍 **Sentiment Analysis**

  * Classifies reviews as Positive, Negative, or Neutral

* 😊 **Emotion Detection (AI-based)**

  * Detects emotions like Joy, Anger, Sadness, etc.

* 📊 **Interactive Dashboard**

  * Built using Streamlit
  * Dynamic filtering by cuisine

* 🍽️ **Cuisine-Based Insights**

  * Compare how people feel about different foods

* 📝 **Real-Time Review Analyzer**

  * Enter your own review and get instant sentiment + emotion

---

## 🧠 Tech Stack

* **Python**
* **Streamlit** (Dashboard)
* **Pandas** (Data Processing)
* **TextBlob** (Sentiment Analysis)
* **Transformers (HuggingFace)** (Emotion Detection)

---

## 📊 Dataset

This project uses a **cleaned subset of the Yelp Open Dataset**, containing:

* Food reviews
* Cuisine type
* Location
* Ratings

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/yourusername/food-emotion-dashboard.git
cd food-emotion-dashboard
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
cd app
streamlit run app.py
```
